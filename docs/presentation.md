# Presentation: My Git Journey

## Project: ZenTask - AI-Assisted Task Manager

### 1. The Collaboration
- **AI Tool**: Antigravity (Gemini 1.5 Pro)
- **Role**: AI acted as a pair programmer, drafting the initial HTML/CSS/JS logic and assisting with code refactoring and project organization.

### 2. Initialization & Setup
- **Command**: `git init`
- **Action**: Created the local repository and established the directory hierarchy.
- **Organization**:
  - `/app`: Functional code.
  - `/docs`: Documentation and Cheat Sheet.
  - `/assets`: Media.
  - `/archive`: Legacy code management.

### 3. Development Workflow
- **Initial Commit**: `git add .` and `git commit -m "Initial commit"`
- **Branching**: Created `feature-update` to implement developer credits safely without affecting the main codebase.
- **Command**: `git checkout -b feature-update`

### 4. Branch Management
- **Merge**: Successfully merged `feature-update` into `main` after verifying the new functionality.
- **Command**: `git merge feature-update`
