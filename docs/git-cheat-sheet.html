Hoja de Referencia de Git

Comandos esenciales para el Proyecto ZenTask & Más

📁 Configuración e Inicio

git init Inicializa un nuevo repositorio Git en la carpeta actual.
git clone [url] Descarga un proyecto existente desde un servidor (GitHub).
git status Muestra el estado de los archivos (nuevos, modificados, etc).

💾 Guardando Cambios

git add [archivo] Prepara un archivo para el siguiente commit (Staging).
git add . Prepara TODOS los archivos modificados.
git commit -m "mensaje" Crea un punto de control con una descripción de los cambios.

🌿 Ramas (Branching)

git branch Lista todas las ramas disponibles en el repositorio.
git checkout -b [nombre] Crea una rama nueva y salta a ella inmediatamente.
git checkout [nombre] Cambia entre ramas existentes.
git merge [nombre] Une los cambios de la rama [nombre] a la rama actual.

🔍 Historial y Utilidades

git log --oneline Lista los commits en una sola línea para mayor claridad.
git log --graph --all Muestra el árbol visual de todas las ramas y commits.
tree /f Muestra la estructura de archivos en consola (Windows).
