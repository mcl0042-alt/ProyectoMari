# ZenTask: AI-Assisted Development & Git Mastery

## 🎯 Project Objective
ZenTask is a modern, web-based task management application developed as part of the **AI-Assisted Development and Git Mastery** assignment. This project showcases the synergy between human developers and Generative AI, while demonstrating professional Git version control workflows.

## 🚀 Key Features
- **Modern UI**: A premium glassmorphic design with smooth animations.
- **Persistent Storage**: Uses `localStorage` to keep your tasks safe across sessions.
- **AI-Driven Logic**: Core logic drafted and optimized with AI assistance.
- **Git Documentation**: Complete history of initialization, branching, and merging.

## 🛠️ Technical Stack
- **Structure**: HTML5 Semantic markup.
- **Styling**: Vanilla CSS3 (Custom Design System).
- **Logic**: Modern JavaScript (ES6+).
- **Version Control**: Git CLI.

## 📖 Git Workflow Summary
1.  **Initialization**: Repository setup using `git init`.
2.  **Tracking**: Regular commits to document progress.
3.  **Branching**: Feature development in `feature-update` branch.
4.  **Merging**: Integration of features into the `main` branch.
